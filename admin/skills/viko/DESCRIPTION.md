# Viko

Skill resmi buatan Viko sendiri (bukan bawaan Hermes).

**Konvensi** (berlaku untuk semua skill resmi Viko):
- **Folder category:** `viko/` — semua skill Viko ditaruh di `admin/skills/viko/{skill}/SKILL.md`
  (Hermes menurunkan category dari folder, jadi semuanya tampil di kategori **viko** di dashboard).
- **Prefix nama:** `viko-` — field `name` di frontmatter selalu `viko-<nama-skill>`
  (mis. `viko-onboarding`).
